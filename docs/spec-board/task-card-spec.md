# spec-board - タスクカード仕様（フロントエンド）

> **機能**: [spec-board](./index.md)
> **ステータス**: 下書き

## 概要

ボード上の各タスクを表すカード型UI要素の表示仕様、詳細パネル、およびタスク作成・編集フォームを定義する。

## コンポーネント

| コンポーネント | 種別 | 説明 | 振る舞い |
|:-------------|:-----|:-----|:---------|
| タスクカード | カード | タイトル・優先度・ラベル・サブIssue進捗を表示する小型カード | ドラッグ可能。クリックで詳細パネルを開く |
| 詳細パネル | スライドオーバー | タスクの全情報を表示・編集できるサイドパネル | カードクリックで右側からスライドイン |
| タスク作成フォーム | モーダル | 新規タスクの入力フォーム | カラムの「+ 追加」ボタンから開く |
| 優先度バッジ | バッジ | 優先度を色で示す小型ラベル | High: 赤、Medium: 黄、Low: 青。未設定時はバッジ非表示 |
| ラベルタグ | タグ | カテゴリやカスタムラベルを示すタグ | ラベルごとに異なる背景色 |
| サブIssue進捗バー | プログレスバー | 子タスクの完了状況を示す水平バー | 完了数/全数を色で示す。子タスクがない場合は非表示 |
| サブIssueリスト | リスト | 子タスクのタイトルとステータスの一覧 | カードに折りたたみ表示。クリックで子タスクの詳細パネルを開く |
| リンクリスト | リスト | 関連タスクのタイトルとステータスの一覧 | 詳細パネルに表示。クリックで遷移 |

## タスクカード表示

### カードレイアウト

```
┌──────────────────────┐
│ [優先度] タイトル      │
│                      │
│ [ラベル1][ラベル2]     │
│                      │
│ ▶ サブIssue (2/5)    │
│  ├ ✓ 子タスク1        │
│  ├ ○ 子タスク2        │
│  └ ...               │
│ [████████░░] 2/5     │
└──────────────────────┘
```

- サブIssueがない場合、サブIssueリストと進捗バーは非表示
- サブIssueリストはデフォルト折りたたみ。▶ クリックで展開/折りたたみ
- リスト展開時は直接の子タスクのみ表示（孫タスクは子タスクの詳細で確認）
- 進捗バーの完了数は、全階層の子孫タスクのうちステータスが最終カラムのものをカウント

### 表示データ

| フィールド | ソース | フォーマット | 空の場合 |
|:----------|:-------|:------------|:---------|
| タイトル | フロントマター `title` | そのまま表示 | ファイル名を表示 |
| 優先度 | フロントマター `priority` | バッジとして表示（High/Medium/Low） | バッジ非表示 |
| ラベル | フロントマター `labels` | タグとして横並びに表示 | タグ非表示 |
| サブIssue進捗 | 子タスクの `status` を集計 | プログレスバー + 「完了数/全数」 | サブIssueセクション非表示 |
| サブIssueリスト | 直接の子タスクの `title` と `status` | チェックアイコン付きリスト | サブIssueセクション非表示 |

## 詳細パネル

### レイアウト

```
┌────────────────────────────┐
│ [×]                        │
│                            │
│ タイトル（編集可能）         │
│                            │
│ ステータス: [Todo ▼]        │
│ 優先度:    [Medium ▼]      │
│ ラベル:    [ラベル1][+追加]  │
│ 親タスク:  タスク名 →       │
│                            │
│ ──── サブIssue (2/5) ──── │
│ [████████░░]               │
│  ✓ 子タスク1    Done       │
│  ○ 子タスク2    Todo       │
│  ○ 子タスク3    In Progress│
│          [+ サブIssue追加]  │
│                            │
│ ──── 関連タスク ────────── │
│  ○ 関連タスク1   Todo  [×] │
│  ○ 関連タスク2   Done  [×] │
│          [+ リンク追加]     │
│                            │
│ ──────────────────────     │
│                            │
│ 説明（Markdown表示）        │
│ ここにmdファイルの本文を     │
│ レンダリングして表示する      │
│                            │
│ ──────────────────────     │
│ ファイルパス: tasks/xxx.md  │
│                            │
│        [削除]              │
└────────────────────────────┘
```

### ユーザー操作

| 操作 | トリガー | 振る舞い | 遷移先 |
|:-----|:--------|:---------|:-------|
| パネルを開く | タスクカードをクリック | 右側からスライドインで詳細パネルを表示 | - |
| パネルを閉じる | ×ボタンクリック / Escキー / パネル外クリック | スライドアウトでパネルを非表示 | - |
| タイトル編集 | タイトル部分をクリック | インライン編集モードに切り替え。Enterで確定、Escでキャンセル。確定時にmdファイルを更新 | - |
| ステータス変更 | ドロップダウン選択 | フロントマターの `status` を更新し、ボード上のカードも移動 | - |
| 優先度変更 | ドロップダウン選択 | フロントマターの `priority` を更新 | - |
| ラベル追加 | 「+ 追加」ボタンクリック | ラベル入力フィールドを表示。入力確定でフロントマターの `labels` に追加 | - |
| ラベル削除 | ラベルタグの×ボタンクリック | フロントマターの `labels` から削除 | - |
| 親タスクへ遷移 | 親タスク名クリック | 親タスクの詳細パネルに切り替え | 親タスクの詳細パネル |
| サブIssue追加 | 「+ サブIssue追加」ボタンクリック | タスク作成フォームを開く（`parent` に現在のタスクを自動設定） | - |
| サブIssueへ遷移 | サブIssueリストの項目クリック | 該当子タスクの詳細パネルに切り替え | 子タスクの詳細パネル |
| リンク追加 | 「+ リンク追加」ボタンクリック | タスク検索・選択UIを表示。選択したタスクのパスをフロントマターの `links` に追加 | - |
| リンク削除 | リンク項目の×ボタンクリック | フロントマターの `links` から該当パスを削除 | - |
| リンク先へ遷移 | リンクリストの項目クリック | 該当タスクの詳細パネルに切り替え | リンク先タスクの詳細パネル |
| タスク削除 | 「削除」ボタンクリック | 確認ダイアログ表示後、mdファイルを削除。子タスクがある場合は子タスクの `parent` をクリアするか確認 | パネルを閉じる |

## タスク作成フォーム

### フィールド定義

| フィールド | 種別 | 必須 | 初期値 | 説明 |
|:----------|:-----|:-----|:-------|:-----|
| タイトル | テキスト | はい | 空 | タスクのタイトル。ファイル名としても使用（kebab-case変換） |
| ステータス | セレクト | はい | 作成元カラムのステータス | タスクのステータス |
| 優先度 | セレクト | いいえ | なし | High / Medium / Low / なし。「なし」選択時はフロントマターに `priority` を出力しない |
| ラベル | タグ入力 | いいえ | 空 | カンマ区切りで複数入力可能 |
| 親タスク | タスク選択 | いいえ | なし（サブIssue追加時は自動設定） | 親タスクを検索・選択。既存タスクから選択 |
| 説明 | テキストエリア | いいえ | 空 | タスクの詳細説明（Markdown） |

### バリデーションルール

| フィールド | ルール | エラーメッセージ | タイミング |
|:----------|:------|:---------------|:----------|
| タイトル | 空文字不可 | タイトルを入力してください | onSubmit |
| タイトル | 同名ファイルが存在しない | 同じタイトルのタスクが既に存在します | onSubmit |

### 送信時の動作

| アクション | 処理 | 成功時 | 失敗時 |
|:----------|:-----|:-------|:-------|
| タスク作成 | フロントマター付きmdファイルを作成 | モーダルを閉じ、ボードに新カードを表示 | トースト通知でエラーメッセージ表示 |

## エラー表示

| エラーケース | 発生条件 | 表示方法 | ユーザーアクション |
|:------------|:---------|:---------|:----------------|
| ファイル保存失敗 | ディスク容量不足、権限エラー | トースト通知 | ファイルシステムの状態を確認 |
| ファイル削除失敗 | ファイルが既に削除されている、権限エラー | トースト通知 | ボードをリロード |
| フロントマターパースエラー | mdファイルのフロントマターが不正 | 詳細パネルにエラーバナー表示 | mdファイルを手動修正 |
| リンク切れ | `links` や `parent` で指定されたファイルが存在しない | 該当リンク項目に警告アイコン + 「リンク切れ」表示 | リンクを削除するか、ファイルを作成 |
| 循環参照 | 親子関係が循環している | 詳細パネルにエラーバナー「循環参照が検出されました」 | `parent` フィールドを修正 |
